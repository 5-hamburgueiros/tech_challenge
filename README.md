### Observação: para Rodar no modo de Desenvolvimento
Duplicar o ***.env.example*** e colocar como ***.env***


```shell

$  yarn install

```
```shell

$  yarn start:dev

```
  

# Executando Docker

```shell
$  docker compose up --build
```
